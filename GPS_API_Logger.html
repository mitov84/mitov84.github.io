GPS API Logger

Send location data from your android device to specified endpoint.

About

With GPS API Logger you can easily send simple HTTP requests with your device location to specified by you endpoints. GPS API Logger is an android application that can access your device location even when the application is not running.
GPS API Logger is perfect for those who need to keep track of their user location for home automation projects or any other project. It is easy to use and can be set up in no time. With GPS API Logger, you can customize the frequency of location updates.
